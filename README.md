# 🦅 鹰眼 HawkEye — IPO 上市预审 AI 系统

> 以"以果推因"法理演绎为骨架，将创业板发审委决策路径工程化为  
> 可复用的三柱预审方法论（TPCL），辅助 IPO 前的合规风险识别。

---

## 项目背景

以 2023 年成都裕鸢航空（创业板因独立性不足、创新性不足被否）为  
方法论锁定锚点，通过"以果推因"逆向归因，独立研究发审委判断路径，  
提炼出 TPCL 三柱闭环方法论，并进行工程化实现。

---

## TPCL 三柱方法论

| 支柱            | 核心问题       | 法规依据                                   |
| --------------- | -------------- | ------------------------------------------ |
| 支柱一 · 独立性 | 企业能否生存？ | 《注册管理办法》第十二条"独立持续经营能力" |
| 支柱二 · 创新性 | 企业能否发展？ | 创业板"三创四新"定位要求                   |
| 支柱三 · 成长性 | 天花板在哪？   | "成长型创新创业企业"定性要求               |

三柱按"生存 → 发展 → 空间"层级递进，通过经营数据形成自验证闭环。

> **边界说明**：TPCL 是创业板"科创定位与商业模式核查"子模块的  
> 方法论沉淀，不涵盖财务真实性红线、股权合规等其他审核维度。

---

## 裕鸢航空体检结论

| 支柱            | 名称               | 结论                       |
| --------------- | ------------------ | -------------------------- |
| 支柱一 · 独立性 | 独立持续经营能力   | 🔴 极高危——恶性寄生型代工厂 |
| 支柱二 · 创新性 | 科创属性与技术壁垒 | 🔴 绝症——技术护城河极浅     |
| 支柱三 · 成长性 | 持续经营与成长能力 | 🔴 绝症——业绩天花板已触及   |
