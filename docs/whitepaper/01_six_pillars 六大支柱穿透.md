## 创业板 IPO 审核的 6 大法理支柱（全科体检表）

| 支柱   | 名称                   | 别名     | 法律结界                               |
| ------ | ---------------------- | -------- | -------------------------------------- |
| 支柱一 | 独立性                 | 业务防线 | 资产 / 人员 / 财务 / 机构 / 业务独立性 |
| 支柱二 | 板块定位与科创属性     | 特色红线 | 三创四新 / 研发 / 技术先进性           |
| 支柱三 | 持续经营与成长能力     | 未来红线 | 主营稳定 / 成长性 / 经营前景           |
| 支柱四 | 财务真实性与内部控制   | 财务红线 | 报表真实 / 内控有效 / 收入合规         |
| 支柱五 | 主体资格与规范运行     | 合规红线 | 历史沿革 / 董监高合规 / 公司治理       |
| 支柱六 | 募集资金运用与信息披露 | 信披红线 | 募投合规 / 披露真实完整                |

**核心命题**：任何一家 拟上市企业，无论行业、无论规模，其被否原因都不会逃出这 6 大支柱的范畴。
