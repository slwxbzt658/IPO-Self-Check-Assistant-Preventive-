**#** **04 - 微创模型清单与设计原则**

**##** **设计原则**

\1. 结界约束:每个微创模型必须挂载在某个法律支柱之下

\2. 高内聚低耦合:支柱间通过 API 调用而非数据耦合

\3. 数据可得性分层:仅在区域 A 实现可推理的探针,深度模型放区域 C
